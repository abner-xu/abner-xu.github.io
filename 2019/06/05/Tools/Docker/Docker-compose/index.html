Docker-Compose详解

工具 Docker Docker 2019/06/05

本文于374天之前发表，文中内容可能已经过时。

Compose中有两个重要的概念：

  • 服务(service)：一个应用的容器，实际上可以包括若干个运行相同镜像的容器实例。
  • 项目(project)：由一组关联的应用容器组成的一个完整的业务单元，在docker-compose.yml文件中定义。
