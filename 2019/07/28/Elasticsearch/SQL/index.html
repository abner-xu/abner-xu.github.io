SQL语法

查询所有

GET /Product/cpu/_search

单条件匹配和排序

GET /Product/cpu/_search

查询指定的字段

GET /Product/cpu/_search

filter数据过滤

GET /Product/cpu/_search
例如：cpu名称包含amd,且价格大于1000

full-text search 全文检索

GET /Product/cpu/_search
例如：desc字段内容去被倒排索引一一检索，包含任意词条的数据会被反驳

phrase search 短语检索

GET /Product/cpu/_search
例如：desc字段内容必须全匹配短语chaopin pianyi

highlight search 高亮检索

GET /Product/cpu/_search
例如：desc字段内容必须全匹配短语chaopin pianyi
